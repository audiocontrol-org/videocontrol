# Text Overlay Generator

**Status:** Planning
**Branch:** `feature/text-overlay`
**Milestone:** [Week of Feb 3-7](https://github.com/audiocontrol-org/videocontrol/milestone/1)

## Overview

Migrate the text overlay generator from oletizi/video-control into the videocontrol monorepo. This tool generates professional text overlay videos (titles, lower-thirds, callouts, code blocks) from YAML definitions, rendering to ProRes 4444 with alpha channel for compositing.
